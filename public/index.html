Free Code Camp Image Search Abstraction Layer

This is part of the requirements for the Free Code Camp Backend Development Certificate

User stories:

  1. I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  2. I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  3. I can get a list of the most recently submitted search strings.
Example usage:

To search for images of 'extincted species', returning 10 results with an offset of 10, do:
https://imsearch-dm-freecodecamp/api/imgsearch/extinct species?offset=10&count=10

To browse the 10 most recent queries, do: https://imsearch-dm-freecodecamp/api/latest/imgsearch
